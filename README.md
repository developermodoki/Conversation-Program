# Conversation-Program
これは、Pythonのinput機能のテストファイルです。テストファイルなので参考までにお願いします。
# 使い方
まず、「今日はいい天気ですね。」と聞かれます。1が「はい」で、2が「いいえ」です。1と答えると「そうですよねー。」と表示されます。2と答えると「そうですか...残念です」と表示されます。
次に、「今日の気分はどうですか？」と聞かれます。1が、「けっこういいです」で、2が「良くないです」です。1と答えると「そうですか。最高の一日にしてください。」と表示されます。2と答え
ると、「そうですか...残念です。この一日でリフレッシュできるように頑張ってください。」と表示されます。
# 注意事項
このファイルを実行するには、Python 3.8の環境が必要です。Python3.8の環境を用意していただくようお願いします。
